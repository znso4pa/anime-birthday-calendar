# 🎂 二次元生日查询

> 一个很轻的二次元角色生日查询工具  
> **只收录了极小部分角色**（约 30+），主要是我自己喜欢的

## ✨ 能做什么

- 按角色名查生日（支持中文 / 罗马音）
- 按日期查当天过生日的角色
- 自动显示「今天谁生日」
- 可以自己添加角色（存在你浏览器的本地存储里）

1. 打开网页
2. 直接搜就行
